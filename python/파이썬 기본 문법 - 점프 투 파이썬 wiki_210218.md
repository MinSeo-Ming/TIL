```python
 filter 되게 유용한데 쓰기 어려울거 같다...!
 쓰는 방법은 함수이름 , 반복 가능한 자료형 요소
- 방법 1
def positive(l):
    result = []
    for i in l:
        if i > 0:
            result.append(i)
    return result
print(positive([1, -3, 2, 0, -5, 6]))
- 방법 2
def positive(x):
    return x > 0
print(list(filter(positive, [1, -3, 2, 0, -5, 6])))
- 방법 3 람다식을 사용한다!! -> 여기 람다식으로!```
